安裝教學

第一次使用大約需要 5-10 分鐘，之後每次只要雙擊啟動

1

下載 CardForge

點擊上方「免費下載」按鈕，會下載一個 ZIP 壓縮檔。下載完成後，在「下載」資料夾找到它，按右鍵選「解壓縮全部」。

2

安裝 Python（第一次才需要）

CardForge 需要一個叫「Python」的免費程式來執行。如果你之前從來沒裝過，請先去安裝。

➡ 點此前往 Python 官方下載頁面

⚠ 重要！ 安裝畫面最下方有一個勾選框寫著「Add Python to PATH」，一定要打勾！沒勾的話程式會無法啟動。勾完之後點「Install Now」就好。
3

啟動 CardForge

打開剛才解壓縮的資料夾，找到 start.bat 這個檔案，雙擊它。

Windows 可能會跳出安全性警告
如果看到「Windows 已保護您的電腦」的藍色畫面，這是正常的。點「更多資訊」，再點「仍要執行」即可。

會出現一個黑色視窗（這是正常的，不要關掉它），等幾秒鐘後，你的瀏覽器會自動打開 CardForge。

如果瀏覽器沒有自動開啟，請手動在網址列輸入 localhost:5001 再按 Enter。

在終端機裡，複製貼上以下指令（把路徑改成你解壓縮的位置），然後按 Enter：
